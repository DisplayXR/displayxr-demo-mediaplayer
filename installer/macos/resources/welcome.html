Stereo Media Player

A lightweight stereo media player for glasses-free 3D displays, built on the DisplayXR OpenXR runtime via Vulkan. Plays already-stereo images (side-by-side .jpg / .png) and video (side-by-side H.264 / H.265 / AV1), routing each half to its eye view with aspect-correct letterboxing.

Once installed, the app lands at /Applications/Stereo Media Player.app. Launch it to see a bundled sample stereo image; pass a media file on the command line to play your own. Hardware video decode (VideoToolbox) is used when available, with a software fallback.

Prerequisite: the DisplayXR runtime must be installed first. Download DisplayXR-Installer-*.pkg from the displayxr-runtime releases page. The demo bundles its own copy of MoltenVK, the OpenXR loader, and FFmpeg, but discovers the actual runtime through the system's OpenXR configuration registered by the runtime installer.
